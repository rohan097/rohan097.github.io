Projects

Seizure Prediction

Made a model that could predict the onset of an epileptic seizure. It used iEEG signals as an input and classified it as an interical signal ( normal signal) or as a preictal signal ( just before the onset of a seizure.)

Features such Fast Fourier Transforms, Eigen Values and Correlation Coefficients were extracted from the input and given to a Random Forest comprising of 3000 trees. An accuracy of 88% was obtained. Future work involves using deep learning to classify the images based on the spectrogram of the signals.

  • Language: Python
  • Library: sk-learn
  • GitHub

pScan

Created an Android app called pScan to diagnose whether the user has Parkinson's Disease by analysing the user's voice.

The user was asked to speak a few vowels for 5 seconds. The recording of the voice was sent to a server for analysis. From the speech, 26 features were extracted such as shimmer, jitter, harmonicity. These features were then fed to a random forest regressor, to predict the UPDRS (Unified Parkinson's Disease Rating Scale) value. This value was then given to a neural net along with the 26 features to classify whether the user had Parkinson's disease. The app gave a accuracy of 91%. I worked on the signal processing to extract the features and also on the Random Forest.

  • Language: Python
  • Libraries: sk-learn, oct2py, Praat, MATLAB
  • GitHub
