Projects

pScan

pScan is an android application to identify whether a user has Parkinson's Disease. It uses a non-invasive technique of analysing the voice. From the voice, over 20 features are extracted. These features are used to diagnose the disease. It is also capable of indicating where the user lies on the UPDRS.

This project was submitted at the Code2Create Hackathon conducted by ACM-VIT. My team won the first prize.

I worked on the processing the speech signal, and the neural network required for the prediction.

  • Tech Stack: Python, Scikit-learn, Octave, Praat, MongoDB
  • Website

Arbitrium

Arbitrium is a B2B dashboard for DHL. It allows DHL to analyse and predict the probability of winning a new client, and the churn rate. It provides useful insights for the operator, such as country wise performance, KPIs, and so on. It analyses social media to understand how a company is performing.

This project was submitted at the #UnitedByHCL Hackathon held at Old Trafford - Manchester, UK. My team secured the third place.

I worked on the prediction model, and certain aspects of the backend.

  • Tech Stack: Python, Scikit-learn, TensorFlow, HTML5, CSS, Bootstrap, MongoDB
  • Website

Smart Traffic Light System

This is a completely automated traffic light system. Unlike conventional systems which rely on fixed duration signals, this system has a more dynamic approach. Through the use of cameras placed at intersections, the number of cars are identified. An algorithm decides the most efficient timings for each lane.

  • Tech Stack: Python, OpenCV, Raspberry PI

Epileptic Seizure Prediction

A system to predict the onset of epileptic seizures was created. It uses iEEG recordings. Features such as Eigen Values, and Spectral Coefficients were used to build the predictive model.

  • Tech Stack: Python, Scikit-learn

Provectus

Provectus is an android application to calculate the insurance premium for a new client seeking insurance. In general, to calculate the premium, the various risks associated with the customer have to be assessed. To do this, several metrics such as smoking patterns, lifestyle, etc were used. One of the main features of the app was to identify the BMI of the user, by only analysing the face.

This project was submitted at the FinTech Valley Hackathon conducted by the Govt. of Andhra Pradesh. We won the "Best Use Case under Health Management and Reporting".

I worked on calculating the BMI from the face. In order to do this, a total of 7 ratios were calculated from the face. This gave a general indication the health of the body. A model was then trained to predict the BMI, with these ratios as input.

  • Tech Stack: Python, Scikit-learn, OpenCV, DLib, MongoDB
