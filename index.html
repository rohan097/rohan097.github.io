Projects

Seizure Prediction

A model to predict the onset of an epileptic seizure. It uses iEEG signals as an input and classifies it as an interical signal (normal signal) or as a preictal signal (just before a seizure).

Features such Fast Fourier Transforms, Eigen Values and Correlation Coefficients are extracted from the input and given to a Random Forest comprising of 3000 trees. An accuracy of 88% was obtained. Future work involves using Deep Learning to classify the images based on the spectrogram of the signals.

  • Language: Python
  • Library: sk-learn
  • GitHub

p-Scan

An Android app called p-Scan to diagnose whether the user has Parkinson's Disease by analysing the user's voice.

The user speaks a few vowels for five seconds. The recording of the voice is sent to a server for analysis. From the speech, 26 features are extracted such as shimmer, jitter, harmonicity. These features are then fed to a Random Forest Regressor, to predict the UPDRS (Unified Parkinson's Disease Rating Scale) value. This value is then given to a feed-forward neural net along with the 26 features to classify whether the user had Parkinson's disease. The app gave an accuracy of 91%. I worked on the signal processing to extract the features and also on the Random Forest.

  • Language: Python
  • Libraries: sk-learn, oct2py, Praat, MATLAB
  • GitHub
  • Website

Automated Traffic Control

A system to control traffic lights at an intersection based on the number of vehicles in a particular lane.

The video feeds from four cameras placed at optimal locations are given to a Raspberry Pi. Using image processing, the number of vehicles are extracted for each lane. The value is then given to a signal control algorithm designed by my team. This algorithm calculates the duration a lane should have a green light based on the density.

  • Language: Python
  • Libraries: OpenCV-3
  • GitHub
